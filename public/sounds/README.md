# Sound Files

Place your custom sound effect files here.

## Supported Formats
- MP3
- WAV
- OGG

## Default Sound

The default `message.mp3` file should be a short notification sound (0.5-1 second).

You can replace it with your own sound file, or add additional sounds for different events:
- `message.mp3` - Default message sound
- `superchat.mp3` - Super chat notification
- `moderator.mp3` - Moderator message sound

## Free Sound Resources

You can find free sound effects at:
- [Freesound.org](https://freesound.org)
- [Zapsplat](https://www.zapsplat.com)
- [Mixkit](https://mixkit.co/free-sound-effects/)

## Creating Your Own

You can create a simple beep sound using audio editing software or online tools like:
- Audacity (free, cross-platform)
- Online Tone Generator: [onlinetonegenerator.com](https://onlinetonegenerator.com)
